◆ load_clusters()

int load_clusters ( char *  filename,
struct cluster_t **  arr 
)

Loads objects from file. Creates cluster for each object and saves it to array of clusters. Allocates space for array of all the clusters and saves the pointer to the first element.

Parameters
filename- file from which the data are loaded
arr- array of clusters
Returns
returns number of objects loaded, if successful
returns -3 if file couldn't be loaded
returns -4 on fgets error
returns -6 on get_count function Error
returns -9 if error in objects file
returns -10 if number of objects is incorrect
